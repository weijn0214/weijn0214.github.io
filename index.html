关于这个页面

这是一个用于演示的静态页面，你可以直接修改这里的文字、图片、样式，打造属于自己的页面。

GitHub Pages 支持纯前端静态资源（HTML/CSS/JS/图片/字体等），无需后端服务，免费且稳定。

你点击了按钮！这是简单的 JS 交互效果~

如何修改这个页面

1. 修改文字：直接替换 HTML 中的文字内容即可；

2. 修改样式：调整 <style> 标签内的 CSS 代码，比如颜色、大小、布局；

3. 添加功能：在 <script> 标签内编写 JS 代码，实现更多交互；

4. 替换图片：将图片上传到仓库，修改 img 标签的 src 为相对路径（如 ./images/xxx.png）。
